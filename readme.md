Dwarf
========
tiny microservice framework for typescript


# Start/stop

## Required

 - docker
 - docker-compose
 - make

## Start

```sh
make start
```

![Dwarf start](/_resource/dwarf-start.png)

## Monitor

```sh
make monitor
```

![Dwarf monitor](/_resource/dwarf-monitor.png)

## Stop

```sh
make stop
```
